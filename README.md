
Just a demo using [MetalPetal](https://github.com/MetalPetal/MetalPetal) to implement Instagram filters.

## Features

* Instagram like photo picker
* All Instagram filters

## Requirements

* iPhone Device
* iOS 11.0+
* Xcode 9.0+
